## 报表在线预览

Java + OpenOffice + jodconverter + easypoi 实现

具体功能：

- 通过easypoi，将报表模板进行数据填充
- 通过openOffice实现Excel转html，Excel转PDF
- Html在线预览
- Excel，PDF导出
